# Instalação de Wordpress

Será utilizado uma stack com Docker para subir os serviços necessários
## instalação

Utilizar o comando para clonar o repositório

```bash
git clone https://www.git.com/alexeiev/wordpress-traefik.git
```
Use

```bash
docker-compose up -d
```
